# Medimart - Your Online Medical Store

Medimart is a modern e-commerce platform designed to simplify the purchase of medicines, medical equipment, and beauty products. Built using PHP and Laravel, it provides a seamless shopping experience with features like theme customization, a user-friendly interface, and secure transactions.

## Features

### Product Management

*   Organized categories for medicines, medical equipment, and beauty products
*   Product search and filtering system
*   Product reviews and ratings

### User Management

*   Secure login and registration for customers
*   User profile management
*   Order history and tracking

### Order Management

*   Add-to-cart functionality with a smooth checkout process
*   Order tracking from placement to delivery
*   Order cancellation and refund management

### Payment Gateway

*   Integrated payment gateway for safe transactions
*   Support for multiple payment methods

### Theme Customization

*   Switch between light and dark themes for a personalized experience
*   Customizable theme colors and fonts

## Tech Stack

*   Backend: PHP, Laravel Framework
*   Frontend: Blade Templates, Tailwind CSS
*   Database: MySQL
*   Dependencies: Composer, NPM
*   Deployment: Localhost using Laravel’s built-in server or external hosting

## Installation

### Prerequisites

*   PHP >= 8.1
*   Composer
*   Node.js and NPM
*   MySQL or compatible database

### Installation Steps

1.  Clone the repository using Git
2.  Run `composer install` to install dependencies
3.  Run `npm install` to install frontend dependencies
4.  Create a new MySQL database and update the `.env` file with database credentials
5.  Run `php artisan migrate` to create database tables
6.  Run `php artisan serve` to start the development server
